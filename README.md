# 企业OA管理系统 java67

## 项目概述

本项目是一个基于Springboot和Vue的企业OA管理系统，采用JavaEE技术栈进行开发，实现了前后端分离。系统遵循MVC三层架构模式，通过Spring、SpringBoot、Mybatis和Vue等技术的整合，构建了一个高效、稳定的企业办公管理系统。

## 技术栈

- 前端：Vue
- 后端：Springboot、JavaEE
- 数据库：MySQL
- 构建工具：Maven

## 功能模块

- 用户管理
- 部门管理
- 文档管理
- 工作流程
- 请假管理
- 报告生成

## 系统角色

- 系统管理员
- 普通用户
- 部门经理

## 运行环境

- JDK 1.8+
- MySQL 5.7+
- Maven 3.5+

## 部署步骤

1. 准备所需的环境及依赖
2. 使用Maven编译项目
3. 导入MySQL数据库
4. 部署前端Vue项目
5. 部署后端Springboot项目
